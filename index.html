Conhecimentos

HTML

Hypertext Markup Language - linguagem que permite que os usuários crie e estruture seções, parágrafos, cabeçalhos e links para páginas da internet ou aplicações. Assim como a organização e formatação de documentos.

CSS

Cascading Style Sheets - linguagem que é usada para estilizar elementos escritos em uma linguagem de marcação como HTML. Ela separa o conteúdo da representação visual do site.

Design

Profissional qualificado para aplicar os conhecimento técnico em páginas web na internet, de modo a melhorar e adpatar a página ao perfil da persona do negócio, para uma melhor experiência do usuário.

JavaScript

È uma linguagem que permite criar páginas Web interativas, parte essencial dos aplicativos hoje na Web. Ela é uma linguagem interpretada, de script em alto nível com tipagem dinâmica fraca e multiparadigma.

Bootstrap

Bootstrap é um framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web usando HTML, CSS e JavaScript.

Git & Github

Git é um sistema de controle de versão de arquivos. Através deles podemos desenvolver projetos na qual diversas pessoas podem contribuir simultaneamente. E o Github é um serviço web que oferece diversas funcionalidades extras aplicadas ao git.
